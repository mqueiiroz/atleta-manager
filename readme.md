# Sistemas de Gestão de Atletas

Este é um projeto pratico desenvolvido em 7 dias, com o foco em consolidar conceitos (POO) com Python, persistências de dados e boas praticas de desenvolvimento.

## O Desafio

Objetivo é construir um sistema para gerenciar atletas e times, incluindo validações, simulações de partidas e sistemas de log.

### 📅 Cronograma de Desenvolvimento
- [x] **Dia 0:** Configuração de Ambiente, Git e Arquitetura de Pastas.
- [x] **Dia 1:** Criação do Modelo de Atleta (Classes e Atributos).
- [x] **Dia 2:** Gestão de Elenco (Agregação de Objetos).
- [x] **Dia 3:** Persistência em JSON (Leitura e Escrita).
- [ ] **Dia 4:** Validação de Dados (Tratamento de Erros).
- [ ] **Dia 5:** Simulador de Partida (Lógica e Aleatoriedade).
- [ ] **Dia 6:** Sistema de Logs (Histórico em .txt).
- [ ] **Dia 7:** Refatoração Final.

## 🛠️ Tecnologias Utilizadas
* **Python 3.14.0**
* **Venv** (Ambiente Virtual)
* **Git & GitHub** (Versionamento)
* **JSON** (Armazenamento de dados)

## 📁 Estrutura do Projeto

atleta-manager/
